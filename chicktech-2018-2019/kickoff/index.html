Capture The Flag

Description: Learn about how people break into computers and networks and do it in a safe environment. We'll be doing a "capture the flag" exercise where you will participate in ever-more difficult exploits, learning about cybersecurity as you go. As the demand for cybersecurity skills has never been higher, this workshop will introduce you to an strong career choice that requires both technical skills and the ability to work well with others. We'll have professionals who work in the field there to help you.

scp bandit13@bandit.labs.overthewire.org:/ect/bandit_pass/bandit14 ~

I used this line of code to get past the bandit level 13. The key for the next level was stored in /etc/bandit_pass/bandit14 and could only be read by user bandit14. I determined that I had to move the file from the remote server to the localhost so I could use the key to access the next level, as this level does not return an actual password.
